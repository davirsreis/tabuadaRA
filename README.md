## Sobre

O jogo funciona como um quiz, com operações básicas da tabuada. O tempo limite para a resolução das operações é definido conforme a dificuldade escolhida.

## Tecnologias utilizadas

* Aframe 0.1.3
* Express 4.18.1
* Express-handlebars 6.0.6
* Boostrap 5.2.2
